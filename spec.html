사용자 에이전트(유저 에이전트)
사용자가 웹을 브라우징 하기 위해 사용하는 모든 장치를 말합니다. 대표적으로 브라우저가 되겠지만, 리더기, 모바일, 점자표시기 등 웹을 표현할 수 있는 모든 장치가 이에 포함됩니다.
이행자
원문에서는 implementor 라 표현하고 있습니다. 문맥상의 의미로 브라우저 제작사들을 지칭하는 것으로 판단합니다.
메서드
객체에 함께 포장되어 있는 함수입니다. mp3 플레이어를 객체로 표현한다면, 수록된 음악들을 프로퍼티, 그리고 플레이 버튼을 메서드라고 생각하시면 무리가 없습니다.
보험요소fallback contents
원래 나타내려고 했던 요소가 어떤 이유(네트워크 문제, 사용자 에이전트와의 비호환 등)로 인해 로딩에 실패했을 경우를 대비하는 컨텐츠입니다. 대표적으로는 img 에 사용하는 alt 텍스트가 있으며, 그밖에 사용자 에이전트가 동영상 컨텐츠를 지원하지 못했을 경우에 대비한 이미지 등이 있습니다.

Abstract

이 명세서는 월드 와이드 웹의 코어 언어 : 하이퍼텍스트 마크업 랭귀지(HTML 5) 의 5번째 주요 개정판을 정의합니다. 이 버전에서는 웹 프로그래머를 위한 새로운 기능들, 기존의 웹 저작 경향을 분석한 결과에 기초한 새로운 요소들, 그리고 상호작용성을 개선하기 위해 사용자 에이전트들이 준수해야 할 사항들을 명확하게 제시하고 있습니다.

This specification defines the 5th major revision of the core language of the World Wide Web: the Hypertext Markup Language (HTML). In this version, new features are introduced to help Web application authors, new elements are introduced based on research into prevailing authoring practices, and special attention has been given to defining clear conformance criteria for user agents in an effort to improve interoperability.
